Project Description

Linksnip is a fast and reliable URL shortening service that turns long URLs into compact, manageable links.
It supports bulk URL shortening and QR code generation, making link sharing effortless and efficient.

Aim

To build a TypeScript-based application that provides APIs for URL shortening, bulk processing, QR code creation, and redirection using a modern Node.js stack.

Tech Stack

Frontend: Vite, TypeScript, TailwindCSS
Backend: Node.js, Express.js, TypeScript
Database: PostgreSQL / MySQL (via Drizzle ORM)
Features: REST APIs, Bulk Shortening, QR Code Generation
Deployment: Docker, Replit
Testing: Postman
